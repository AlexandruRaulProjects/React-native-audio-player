# react-native-audio-player
This is an Audiobook app for creating and playing personalized audios.

# How to run it
## In order to be sure it will work, follow these desribed **steps** bellow in this order, as well as for the **file-reader-app**

## Prerequisites:

1. Nodejs: `https://nodejs.org/en`
2. Git: `https://git-scm.com/downloads`
3. A smartphone with Android installed as OS
4. Expo Go app from Play Store mobile application: `https://expo.dev/go?sdkVersion=51&platform=android&device=true`
5. To access this GitHub project: `https://github.com/AlexandruRaulProjects/file-reader-app` and follow these steps 
6. Firebase project configuration parameters

## To build and run this app:

1. Clone this repo
`git clone <repo-link>`
### Example: `git clone https://github.com/AlexandruRaulProjects/React-native-audio-player.git`

2. Go to **root project folder** (where you will find the App.js, package-lock.json, package.json, ... files)
`cd <where you cloned this repo in the previous step>`

3. Initialize all the required npm packages
`npm init`

4. Run it by using any terminal (e.g. PowerShell, CMD, ...)
`npm start`

5. Install Expo Go, by scanning the provided QR code from the link sent above, using your phone

6. Scan the QR code from the terminal, where the app is running 

7. A `env.js` file which includes Firebase configuration keys, ...
